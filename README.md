# Teacher Photo Editor

A simple browser-based photo editor for teachers to name, crop, and rotate photos of student projects.

## How to Use

1. Open `index.html` in any modern browser (Chrome, Firefox, Safari, Edge)
2. Upload photos by dragging and dropping or clicking "Browse Files"
3. Click a thumbnail to select a photo for editing
4. Use the toolbar to:
   - **Name** — type a name in the text field (used as the filename on save)
   - **Rotate** — click the left/right rotate buttons for 90-degree rotation
   - **Crop** — click Crop, draw a selection on the photo, then click "Apply Crop"
   - **Save** — download the current photo as PNG or JPEG
   - **Save All** — download all photos at once

No installation or internet connection required.
